可以使用哪个MySQL函数找出字符串的长度（以位为单位）？

  • 首页|
  • 文章分类|
  • 正文

MySQL BIT_LENGTH()字符串函数用于获取字符串的长度（以位为单位）。

语法

BIT_LENGTH(Str)

在这里，Str，BIT_LENGTH()函数的参数，是要获取其BIT_LENGTH值的字符串。Str可以是字符串或数字字符串。如果是字符串，则必须用引号引起来。

示例

mysql> Select BIT_LENGTH('New Delhi');  +-------------------------+ | BIT_LENGTH('New Delhi') | +-------------------------+ | 72                      | +-------------------------+  1 row in set (0.00 sec)  mysql> select BIT_LENGTH(123456);  +--------------------+ | BIT_LENGTH(123456) | +--------------------+ | 48                 | +--------------------+  1 row in set (0.00 sec)
